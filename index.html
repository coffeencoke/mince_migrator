Mince Migrator

Database migration library for ruby applications

Download Download View Mince Migrator on GitHub

1. Install

$ gem install mince_migrator

2. Use

$ mince_migrator create Load admin users
# Implement your migration by opening db/migration/load_admin_users.rb
$ mince_migrator run Load admin users

3. Learn more

Bootstrap your application

You can bootstrap your app by adding a file located at config/mince_migrator.rb in your app's root directory.

Rails

For Rails you will want to add the following:

# This will load the correct database and mince interface
# for us to use to run our migrations.
environment = ENV['RAILS_ENV']||'development'

require_relative 'environment'

Other Commands

# Revert your single migration
$ mince_migrator revert "Load admin users"

# Run all migrations that have not yet been ran
$ mince_migrator run_all

# View all migrations
$ mince_migrator list # *My favorite!*

# Check the status of a migration
$ mince_migrator show "Load admin users"

# View all available commands
$ mince_migrator --help

Mince

Although you do not need to use Mince in order to use Mince Migrator, however, it makes sense to do so. But if you like this migration tool more than your current ORMs migrator, than you can use this in tandem with your other ORM.
